# Frage-Herkunft zur Steuerung der Wiederverwendung nutzen

Fragen haben eine Herkunft: entweder Bereich oder Standort. Bereichsfragen dürfen nur in Rallyes dieses Bereichs verwendet werden, während Standortfragen in Rallyes des ganzen Standorts verwendet werden dürfen; standortübergreifende globale Fragen gibt es nicht.
