# Frage-Herkunft zur Steuerung der Wiederverwendung nutzen

Fragen haben eine Herkunft: entweder Abteilung oder Standort. Abteilungsfragen dürfen nur in Rallyes dieser Abteilung verwendet werden, während Standortfragen in Rallyes des ganzen Standorts verwendet werden dürfen; standortübergreifende globale Fragen gibt es nicht.
