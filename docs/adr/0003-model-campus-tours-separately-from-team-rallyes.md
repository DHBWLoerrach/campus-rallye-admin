# Campus-Touren getrennt von Team-Rallyes modellieren

Campus-Touren sind keine Team-Rallyes: Sie haben keine Teams, keine gespeicherten Antworten, keine Spielzeit, keinen Rallye-Code und kein Rallye-Ende. Sie verwenden nur den Lebenszyklus aktiv/inaktiv und werden getrennt von konkreten Team-Rallye-Durchläufen modelliert.
