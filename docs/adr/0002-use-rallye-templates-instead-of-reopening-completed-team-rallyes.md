# Rallye-Vorlagen statt Wiederöffnen beendeter Team-Rallyes verwenden

Beendete Team-Rallyes bleiben finale konkrete Durchläufe mit eigenen Teams, Antworten, Spielzeiten und Ergebnissen. Wiederverwendung erfolgt, indem aus einer Rallye-Vorlage eine neue Team-Rallye erstellt wird, damit historische Durchlaufdaten nicht mit künftigen Durchläufen vermischt werden.
